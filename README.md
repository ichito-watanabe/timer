# SPI Timer

SPI勉強用のストップウォッチ＆カウントダウンタイマー。

## URL

**GitHub Pages:** https://ichito-watanabe.github.io/timer/

> GitHub Pages が有効になっていない場合は、リポジトリの Settings → Pages → Branch を `master` に設定する。

## 起動方法

### ブラウザで直接開く（一番簡単）

`index.html` をダブルクリックしてブラウザで開くだけ。

### ローカルサーバーで起動する（推奨）

```bash
# Python がある場合
python -m http.server 8080

# Node.js がある場合
npx serve .
```

起動後、ブラウザで `http://localhost:8080` を開く。

## キーボードショートカット

| キー | 操作 |
|------|------|
| `Space` | スタート / ストップ |
| `Enter` | ラップ / 次問 |
| `O` | 直近ラップを ○（正解）にする |
| `X` | 直近ラップを ✗（不正解）にする |
| `Delete` / `Backspace` | リセット（採点済みなら自動保存） |

## 機能

### ストップウォッチモード
- ラップタイム記録（最速・最遅を色分け表示）
- 分野を選んで○/✗採点 → 成績タブに保存可能

### カウントダウンモード
- プリセット：30秒 / 1分 / 2分 / 英語20分 / 言語30分 / 非言語35分
- +10秒 / +30秒 / +1分 / +5分 ボタンで秒数を自由に足し算して設定（CLR でリセット）
- 残り25%でオレンジ、残り10%で赤フラッシュ
- 時間切れ後もマイナス表示で継続
- 問題数・平均ペースをリアルタイム表示
- 分野を選んで○/✗採点 → 成績タブに保存可能

### 成績タブ
- 分野別正答率を一覧表示（低い順に並ぶ）
- 緑：75%以上 / オレンジ：50〜74% / 赤：50%未満
- データはブラウザのLocalStorageに保存
